# TermuxAdder
Script add member telegram lewat termux

### Langkah pertama

```
pkg update && pkg upgrade -y
pkg install git -y
git clone https://github.com/kenkansaja/TermuxAdder
pkg install python -y
cd TelegramScraper
chmod +x * && python3 setup.py
```
